# Bouboum Ranking

Statistiques du classement Bouboum.

## Installation

```bash
docker-compose up -d nodejs redis db
```
Certbot et Webserver sont des services utilisés pour le déploiement en production.

## Utilisation
```bash
docker-compose exec app npm run webpack:dev
```
